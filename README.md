# 💰 배당 플래너

간단한 배당주 관리 및 재투자 시뮬레이션 도구

## 🚀 기능

- 📈 배당주 종목 관리 (추가/삭제)
- 📊 월별 배당금 차트
- 🎯 재투자 시뮬레이터
- 💡 목표 금액 도달 기간 계산

## 🏃‍♂️ 실행

```bash
npm install
npm run dev
```

## 📁 구조 (경량화됨)

```
src/
├── DividendApp.tsx    # 메인 앱 (모든 기능 통합)
└── main.tsx          # 엔트리 포인트
```

## 📊 지원 종목
